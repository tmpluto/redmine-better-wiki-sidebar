# Redmine Better Wiki Sidebar

# This is Firefox version:
FOR FIREFOX & FIREFOX BASED BROWSERS:
1. clone the source code and `npm i && npm run build`
2. go to `about:debugging#/runtime/this-firefox`
3. Click **Load Temporary Add-on...** and choose `manifest.json` file from the `dist` folder
